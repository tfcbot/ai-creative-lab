# research-ads-meta

Search Meta Ad Library by keyword or company, fetch ad details, and report what's running.

## Providers

- **Scrape Creators** — https://docs.scrapecreators.com/llms.txt
- **Gemini** — https://ai.google.dev/gemini-api/docs/video-understanding

## Steps

1. Keyword/company search Meta Ad Library
2. Fetch ad details

## Input

- keyword or company

## Output

- active ads table

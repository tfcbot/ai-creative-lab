# add-captions

Upload a video to Captions.ai, create a captioning task, poll until complete, and download the result.

## Providers

- **Captions.ai (Mirage)** — https://captions.ai/help/docs/api

## Steps

1. Upload
2. Create captioning task
3. Poll
4. Download

## Input

- video

## Output

- captioned video

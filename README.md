# ai-creative-lab

A marketplace of AI video skills from the **AI Creative Lab** livestream. Install only the skills you want.

## Install the marketplace (once)

In Claude Code:

```
/plugin marketplace add tfcbot/ai-creative-lab
```

## Install a specific skill

```
/plugin install <skill>@ai-creative-lab
/reload-plugins
```

Each skill is its own plugin, so you only get what you pick.

## Browse what's available

```
/plugin
```

Open the plugin manager and go to the **Discover** tab.

## Update

```
/plugin marketplace update ai-creative-lab
```

Pulls new skills published since you last updated.

## Remove a skill

```
/plugin uninstall <skill>@ai-creative-lab
```

## Cross-agent install (Cursor, Codex, OpenCode, etc.)

Skills can also be installed via the [vercel-labs skills CLI](https://github.com/vercel-labs/skills):

```
npx skills add tfcbot/ai-creative-lab --skill <skill>
```

This copies raw SKILL.md files into the agent's skills directory and invokes them flat (`/<skill>` instead of `/<skill>:<skill>`).

## Repo layout

Dual layout. `skills/` is the source of truth. `plugins/` is generated by the sync script for the Claude Code plugin install path.

```
skills/<name>/SKILL.md                    ← source of truth, authors edit here
plugins/<name>/
├── .claude-plugin/plugin.json            ← per-skill plugin manifest (auto-generated on first sync)
└── skills/<name>/SKILL.md                ← copy, regenerated on every sync
.claude-plugin/marketplace.json           ← marketplace catalog (regenerated on every sync)
scripts/sync.ts                           ← run this before committing
```

### Adding or editing a skill

1. Edit or create `skills/<name>/SKILL.md` (only file you author by hand)
2. Run `bun scripts/sync.ts` to regenerate `plugins/` and `marketplace.json`
3. Commit both trees + push

The sync script reads frontmatter from SKILL.md to generate the plugin manifest the first time. On subsequent syncs, it preserves your custom `plugins/<name>/.claude-plugin/plugin.json` and only updates the SKILL.md copy.
